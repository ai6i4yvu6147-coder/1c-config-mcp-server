## Контекст для ИИ

Полная документация и правила проекта находятся в папке `docs/`.

### Начать отсюда

1. `docs/agent-onboarding.md`
2. `docs/README.md` (оглавление)

### Самые важные правила

- **NO_DB_MIGRATIONS**: никаких миграций/конвертаций существующих SQLite БД — базы всегда пересоздаются через `admin_tool`.
- **Whitelist метаданных**: новые типы добавляются точечно по мере необходимости.
- **Тестирование**: проверка функциональности только на рабочем MCP через вызовы его tools (см. `docs/testing-protocol.md`).
